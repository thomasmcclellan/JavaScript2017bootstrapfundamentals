Glyphyicons

Glyphyicons are symbols or icons that are included in Bootstrap. The way you include them in your html is simple:

  • Use a span tag with the following class: "glyphicon glyphicon-hear
  • The first class of "glyphyicon" is the base class; it tells the span class that you will be using an icon.
  • The second class (i.e. glyphicon-heart) points to the specific icon you want to use, in this example, the heart icon
  • You can see the entire glyphicon library on the bootstrapwebsite
